Hoth

Hoth foi a arena de uma batalha entre o Império Galático e a Aliança Rebelde, que tinha instalado uma base provisória, a Base Echo. No entanto, graças aos milhares de droides sondas que haviam espalhados pela galáxia, os imperiais tinham descoberto a base dentro de 1 mês de habitação dos rebeldes no planeta. Embora a Base Echo tenha sido evacuada, a Batalha de Hoth resultou em uma derrota para a aliança depois da elite imperial a Força Blizzard os invadirem.
